# AutoPortalDB

AutoPortalDB to przykładowa baza danych SQL Server dla portalu sprzedaży samochodów.
Zawiera tabele, widoki, zapytania, transakcje, dane testowe oraz konfigurację uprawnień.

## Jak uruchomić
1. Otwórz SQL Server Management Studio (SSMS).
2. Uruchom plik database\AutoPortalDB.sql.
3. Sprawdź poprawność danych przy pomocy zapytaä testowych na końcu pliku.

## Wymagania
- Microsoft SQL Server 2019 lub nowszy
- SQL Server Management Studio (SSMS) lub Azure Data Studio
